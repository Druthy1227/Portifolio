Laboratório de Cibersegurança Web

  • Burp Suite
  • SQL Injection
  • XSS Analysis
  • Web Security

Ambiente controlado de análise e exploração de vulnerabilidades web, com foco em SQL Injection e Cross-Site Scripting (XSS). O projeto contempla a identificação de vetores de ataque, interceptação e modificação de requisições via Burp Suite e a implementação de validações server-side robustas como contramedida. Objetivo: desenvolver o mindset ofensivo para construir sistemas defensivamente melhores.

API REST de Agendamentos

  • Node.js
  • TypeScript
  • PostgreSQL
  • Supabase

API RESTful completa para gerenciamento de agendamentos, desde a modelagem relacional no PostgreSQL com relacionamentos many-to-many até a autenticação via Supabase Auth. O projeto prioriza segurança server-side: validação de entradas, sanitização de dados, controle de permissões por papel (RBAC) e proteção contra ataques de injeção. Arquitetura em camadas seguindo princípios SOLID para máxima manutenibilidade.

Plataforma de Gestão de Tarefas

  • React
  • Node.js
  • Figma
  • Metodologias Ágeis
  • UI/UX

MVP de plataforma full-stack para gestão de tarefas desenvolvido do zero, aplicando metodologias ágeis (Scrum) e planejamento com 5W2H e Metas SMART. O front-end em React foi prototipado no Figma antes da implementação, garantindo consistência de UX. No back-end, a API em Node.js oferece CRUD completo com autenticação JWT, provando que soluções simples, bem arquitetadas, resolvem problemas reais com mínima dívida técnica.
